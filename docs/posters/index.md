# Sample of Static Ecology Infographic Posters

Although we discourage the use of static infographic images in our book,
we want to support teachers that are also printing posters for events
like science fairs where a well-designed poster can draw people into
a booth.  Here are some examples of posters that we
generated.  

## Note on "Fake Data" used by Generative AI

Some of them had a draft that was generated using
a single-shot prompt, where the text-to-image LLM was responsible for
both the layout and generating mock data (synthetic data).  However,
we can see that the numbers were not real.  To correct this we
created new skill that has a two step method.  It generates verified data first and then
calls the text-to-image tools with the verified data.
You can try this skill by installing it in your AI harness and then typing "Describe the verified-infographic-generator skill".

The following static infographic posters were generated by the OpenAI Images 2.0 model
in April 2026.  Prior models had a difficult time with accurate text placement and rendering.
